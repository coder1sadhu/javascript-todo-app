# 📝 JavaScript To-Do App

A modern To-Do List application built with HTML, CSS, and JavaScript. It helps users manage daily tasks with a clean interface and stores data using the browser's Local Storage.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit tasks (Double-click to edit)
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Data saved using Local Storage
- ⌨️ Press Enter to quickly add tasks
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 📂 Project Structure

```
javascript-todo-app/
│── index.html
│── style.css

└── README.md
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/coder1sadhu/javascript-todo-app.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or dependencies are required.
